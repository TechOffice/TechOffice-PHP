# PHP Data Object (PDO)
It is a lightweight, consistent interface for accessing database in PHP. Different database driver would implement the PDO interface and expose database-specific feature.

## Supported Database
* MS SQL Server
* Firebird
* IBM
* Informix
* MySQL
* Oracle
* DB2
* PostgreSQL
* SQLite
* 4D
